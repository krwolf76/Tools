# Tools - V1.0.0 | 서버 설정 도우미, 입퇴장 알림, 플레이어 데이타 수집

## 다운로드
[Tools 다운로드](https://drive.google.com/file/d/1SxmwALzFfsKLzn8BAE_hk0qg9FOrvVO1/view?usp=sharing)
## 미래 계획 -
* 접속 국가 체크
* 코드 최적화
* 왼쪽 이미지 플레이어 아이콘 또는 서버 지정
* 알림 시스템
